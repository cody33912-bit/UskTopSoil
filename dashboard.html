Firebase Configuration Required

To enable real-time automatic updates, you need to create a free Firebase project and add your config details at the bottom of dashboard.html and inside the scripts on index.html / prices.html.

  1. Go to Firebase Console and create a project.
  2. Create a Web App and copy the config object.
  3. Go to Build -> Firestore Database -> Create Database (Start in test mode or set read/write rules).
  4. Replace the placeholder firebaseConfig in the code with your real config.
